Text Area

In this exercise, you will be using Interacto to create a text area and its corresponding ioTextarea interaction.
The command will be a command that changes the text of the text area (undoable).
You will also need to implement an undo and redo functionality like in the previous exercise.
The text interaction stops only when the user makes a pause greater than 1s (timeout customisable).
On each key type, the interaction updates so that the command must be updated using then.
