Text Area

1

In this exercise, you will be using Interacto to create a text area and its corresponding ioTextarea interaction.
The command will be a command that changes the text of the text area (undoable).
You will also need to implement an undo and redo functionality so that the user can revert the color change.
Use Interacto's undoable command functionality to implement this feature.
If you need help, refer to the following documentation: Undoable Commands .
