Image Display

In this exercise, you will use Interacto to create an image carousel with keyboard and mouse interactions.
You will implement commands that change the currently displayed image in response to keyboard inputs, and zoom in and out in response to mouse inputs.
You will need to implement an undo and redo functionality for the commands, similar to the previous exercises.
The image carousel should allow the user to switch between pictures using the left and right arrow keys.
Additionally, the user should be able to zoom in and out using the ctrl key and the mouse wheel.
