Drag and Drop

The goal of this exercise is to understand how to use the drag and drop (DnD) interaction feature provided by Interacto.
On the left side of the screen, you will find a collection of small pictures representing different objects that need to be trashed.
On the right side, there is a bin where the objects should be dropped.
Use the drag and drop functionality to move the object from its initial position to the bin.
The "remove object" command will only be executed if the drag and drop action ends with the object placed inside the bin.
Pay attention to the "canExecute" method, which determines whether the command can be executed based on the successful completion of the drag and drop action onto the bin.
