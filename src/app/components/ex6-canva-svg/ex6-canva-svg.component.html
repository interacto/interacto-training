Canvas

In this exercise, using Interacto with an SVG canvas, you will create a component with the following features:
Double-click: Double-clicking on the canvas creates a predefined-size rectangle (undoable).
Long pressure: Applying long pressure on a rectangle removes it (undoable).
Tree-based undo history: Implement a tree-based undo history on the left, showing command sequences.
Thumbnail history: Commands should generate canvas thumbnails for visualizing changes in the history.
Drag and drop (DnD): Enable DnD functionality on rectangles for moving them, should be (undoable).
Reciprocal DnD: Replace regular DnD with reciprocalDnD.
